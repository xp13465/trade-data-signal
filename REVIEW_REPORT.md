## 二、金融从业者视角

### 2.1 有用且好用的

| 功能 | 评分 | 理由 |
|------|------|------|
| 概览 KPI 卡片 | ★★★★☆ | 一眼看今日情绪/涨跌停/成交额，开盘前必看 |
| 市场宽度（涨跌家数） | ★★★★★ | 最核心的日内情绪指标，比指数更真实 |
| 行业涨跌幅热力图 | ★★★★★ | 板块轮动一目了然，同类产品少见 |
| 指数+买卖点叠加 | ★★★★☆ | 信号和价格同框，验证信号是否合理 |
| 情绪分（0-100） | ★★★★☆ | 量化情绪，比"感觉"靠谱 |
| 冰点热力图 | ★★★★☆ | 多指数情绪共振可视化，抄底逃顶参考 |
| 期货机构持仓 | ★★★★☆ | 机构行为追踪，但刚上线可继续打磨 |

### 2.2 缺失的分析观点

| 缺失项 | 重要性 | 说明 |
|--------|--------|------|
| **涨跌家数比（腾落线）** | 🔴 高 | 已有涨跌家数但没算 AD Line（累计涨跌差），这是判断背离的经典指标——指数涨但 AD 线跌=假突破 |
| **成交额 MA5/MA20 对比** | 🔴 高 | 有成交额但没标注"放量/缩量"，量价关系是技术分析核心 |
| **新高新低家数** | 🟡 中 | NH-NL 是 IBD 核心指标，判断市场内在强度 |
| **板块轮动速度** | 🟡 中 | 行业热力图每天变，但没量化"轮动快慢"——一周内领涨板块换了多少次 |
| **恐贪指数（综合）** | 🟡 中 | 已有6个情绪分，但没合成一个"恐贪指数"（类似 CNN Fear & Greed） |
| **MACD/均线状态** | 🟡 中 | 买卖点信号用到了 MACD 和 MA60，但看板上没展示各指数的均线排列状态（多头/空头排列） |
| **量比（今日成交/5日均量）** | 🟢 低 | 概览快速判断今天是放量还是缩量 |

### 2.3 数据质量问题

| 问题 | 影响 |
|------|------|
| 北向资金停更 2024-08 | 已标注，但概览 KPI 卡片仍显示占位，建议移除或替换 |
| QVIX 1000 停更 2026-03 | 折线图末尾平线，容易误导 |
| 部分行业资金流/换手率缺数据 | 显示"暂无数据"，体验尚可 |

## 三、设计师视角

### 3.1 核心问题：右侧大面积空白

**概览Tab**布局结构：
```
┌──────────────────────────────────────┐
│ KPI 卡片行（6个）         ← 很宽    │
├──────────────────────────────────────┤
│ Spark 指数网格（10个）    ← 很宽    │
├──────────────────────────────────────┤
│ 市场宽度折线（全宽）      ← 很宽    │
├──────────────────────────────────────┤
│ 近6月情绪分（全宽）       ← 很宽    │
├──────────────────┬───────────────────┤
│ 今日买卖点        │ 近期冰点日        │ ← 终于不宽了
├──────────────────┴───────────────────┤
│ 行业热力图（全宽）      ← 很宽      │
└──────────────────────────────────────┘
```

**问题**：大部分图表是全宽的，在 1920px 屏幕上只有中间 800px 有内容，两侧 500+px 空白。尤其是"近6月情绪分"这样的单线折线图，全宽浪费严重。

### 3.2 改进建议

**布局方案A：两列主布局**
```
┌──────────────────────┬──────────────────────┐
│ KPI 卡片（全宽）                             │
├──────────────────────┼──────────────────────┤
│ 市场宽度（左）        │ 情绪分（右）          │
├──────────────────────┼──────────────────────┤
│ 指数 Spark（全宽）                           │
├──────────────────────┼──────────────────────┤
│ 今日买卖点（左）      │ 行业热力图（右）      │
└──────────────────────┴──────────────────────┘
```

**布局方案B：三列 Dashboard**
```
┌────────────┬────────────┬────────────┐
│ 左：情绪KPI │ 中：主要图表 │ 右：信号列表│
│ 情绪分     │ 市场宽度   │ 今日买卖点  │
│ 涨停/跌停  │ 指数Spark  │ 近期冰点    │
│ 成交额     │ 行业热力图 │ 重要提醒    │
└────────────┴────────────┴────────────┘
```

### 3.3 其他设计问题

| 问题 | 建议 |
|------|------|
| **Tab 过多** | 6 个 Tab + 周期选择器挤在顶部，建议合并：概览/大盘(合并A股+港股+全球)/情绪/行业 |
| **ruleBar 每个Tab重复** | 每个Tab顶部都有策略说明，6个Tab看6次。建议改成全局浮动按钮，点击弹出 |
| **颜色不一致** | A股红涨绿跌 ↔ 情绪分红冰点绿过热，同一页面两种配色逻辑，散户容易搞混 |
| **滚动疲劳** | 行业Tab 30+行业+27概念，滚动到手指酸。建议加锚点导航或折叠分组 |
| **KPI 卡片无排序** | 涨停/跌停/成交额/情绪分 随机排列，没有优先级。建议按重要性排序 |
| **期货区过长** | 4张折线图+表格+概览表，占情绪Tab 60%空间。建议折叠或单独Tab |

## 四、散户视角（模拟真实用户）

> 我是一名普通散户，炒股3年，会用同花顺看K线，不太懂技术指标细节。

### 4.1 我觉得有用、好用的

**① 概览页 KPI 卡片** ⭐⭐⭐⭐⭐
"一打开就能看到今天涨了多少家、跌了多少家、涨停多少、跌停多少，跟同花顺的感觉差不多。情绪分那个数字也直观，绿色安心红色紧张。"

**② 行业热力图** ⭐⭐⭐⭐⭐
"这个好！每天哪些板块涨、哪些跌，一眼看出来。以前要翻同花顺行业板块一个个看，这个一张图全搞定。"

**③ 指数+买卖点** ⭐⭐⭐⭐
"图上有红色买点、绿色卖点标记，虽然不知道准不准，但至少有个参考。以前看K线图都是自己瞎猜底和顶。"

**④ 涨停/跌停数据** ⭐⭐⭐⭐
"能看涨停家数趋势，知道市场情绪热不热。涨停家数多的时候做短线成功率会高一些。"

### 4.2 我觉得没用、看不懂的

**① 情绪分"0-100"数字** ⭐⭐
"A股综合情绪分 45 分，是好还是不好？45 和 55 有什么区别？我只看涨跌，这个数字对我来说太抽象了。建议标'偏冷/中性/偏热/狂热'这种文字。"

**② 炸板率/封板率/连板** ⭐
"炸板率是什么？封板率又是什么？我知道涨停，但不知道'炸板'。这些指标应该是给打板族用的，我只买ETF的不需要。"

**③ 换手率分布分位数** ⭐
"换手率>5%家数占比、P25/P50/P75... 看不懂。换手率我知道，但分位数是什么？"

**④ 跨市场综合评分** ⭐
"跨市场综合评分是啥？跟A股情绪分有啥区别？两个都是0-100分，放在一起我看不出区别。"

**⑤ 期货机构持仓** ⭐⭐
"机构净多净空，这个挺有意思的能看到大资金动向。但30/60/120日准确率表格太复杂了，能不能直接告诉我'机构最近准不准'？"

**⑥ 龙虎榜/解禁/IPO** ⭐
"这些都是A股Tab里折叠的，平时根本不会点开看。龙虎榜数据好像也没什么用，我买的是ETF不是个股。"

### 4.3 我希望有的功能

| 功能 | 为什么想要 |
|------|-----------|
| **一句话总结** | "今天市场怎么样？"就像天气预报"今天多云转晴"，给我一个简短结论 |
| **涨跌家数比** | 经常听财经博主说"今天涨跌比1:2，不太好"，这里看不到 |
| **成交量对比** | 想知道今天比昨天放量还是缩量，放量上涨才踏实 |
| **行业强弱排名** | 哪些行业在走强、哪些在走弱，不只是今天涨跌，要看趋势 |
| **大盘位置感** | 现在上证在什么位置？是高位还是低位？能不能给个"便宜/合理/偏贵"的判断 |

## 五、综合优化建议（按优先级）

### P0 — 立即改（体验明显提升）

1. **概览改为两列/三列布局**：解决右侧大面积空白，参考方案B
2. **情绪分加文字标签**：45分="中性偏冷"，18分="冰点"，不用只看数字猜
3. **KPI 卡片排序**：涨停家数→跌停家数→涨跌比→成交额→情绪分→...

### P1 — 近期改（功能增强）

4. **合并全球+H股为"海外"Tab**：减少Tab数量
5. **ruleBar 改为全局浮动按钮**：不每个Tab重复
6. **期货区折叠**：默认显示概览表+准确率摘要，折线图折叠
7. **行业Tab 加锚点导航**：申万行业 / 概念板块 快速跳转

### P2 — 后续迭代（新功能）

8. **新增"涨跌家数比 + 腾落线"**：经典技术指标
9. **新增"成交量对比"**：今日成交 vs 5日均量，放量/缩量标注
10. **新增"大盘位置感"**：指数当前在历史分位数，便宜/合理/偏贵
11. **新增"一句话总结"**：AI 生成今日盘面简报

### P3 — 长期优化

12. **恐贪指数**：合成6个情绪分为一个综合分
13. **板块轮动速度**：量化轮动快慢
14. **新高新低家数**：IBD 核心指标
15. **均线排列状态**：多头/空头排列一目了然
