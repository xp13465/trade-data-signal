# 图表排序优化方案（散户视角）

## 优化原则
- 散户最关心：赚钱效应（涨停）> 大盘温度（涨跌比）> 资金流向 > 情绪 > 技术指标
- 高频决策信息放前面，低频参考信息放后面
- 热力图/全局视图优先于细分图表

---

## 一、市场Tab - A股（`renderAStock`）

### 当前顺序
```
1. 市场宽度（涨跌家数）
2. 涨停/跌停/连板
3. 炸板率/封板率/打板溢价
4. 资金面（北向资金/融资融券）
5. 情绪指数（QVIX/换手率）
6. 换手率分布分位数
7. 换手率>5%家数占比
8. 股息率
[更多] 龙虎榜/解禁/IPO/可转债
[指数区] 各指数K线+买卖点
```

### 建议顺序
```
1. 涨停/跌停/连板          ← 散户最关心赚钱效应
2. 市场宽度（涨跌家数）     ← 大盘整体涨跌比
3. 资金面（北向资金/融资融券）← 聪明钱往哪流
4. 炸板率/封板率/打板溢价    ← 打板生态好坏
5. 情绪指数（QVIX/换手率）  ← 情绪温度
6. 换手率分布分位数         ← 市场活跃度
7. 换手率>5%家数占比        ← 异常活跃股
8. 股息率                   ← 防御性参考，放最后
[更多] 龙虎榜/解禁/IPO/可转债
[指数区] 各指数K线+买卖点
```

### 涉及文件
- `web/app.js` — `renderAStock()` 函数中8个分组折线图的渲染顺序
- `static-site/app.js` — 同上

---

## 二、情绪Tab（`renderSentiment`）

### 当前顺序
```
1. 恐贪指数折线图
2. A股综合情绪分折线图
3. 细分指数情绪分（上证50→沪深300→中证500→中证1000→创业板→科创50）
4. 指数情绪冰点/过热热力图
5. 跨市场综合评分折线图
6. 期货机构持仓模块
```

### 建议顺序
```
1. 指数情绪冰点/过热热力图   ← 一眼全局：哪些指数在冰点/过热
2. 恐贪指数折线图            ← 市场温度计，散户最熟悉的情绪指标
3. A股综合情绪分折线图       ← 综合评分
4. 细分指数情绪分（按散户关注度排序）：
   中证1000/国证2000 → 创业板 → 科创50 → 中证500 → 沪深300 → 上证50
   理由：散户更关注小盘/成长股的情绪波动，大盘股情绪变化慢参考价值低
5. 跨市场综合评分折线图      ← 全球视野，补充参考
6. 期货机构持仓模块          ← 专业参考，放最后
```

### 涉及文件
- `web/app.js` — `renderSentiment()` 函数
- `static-site/app.js` — 同上

---

## 三、行业Tab（`renderIndustry`）

### 当前顺序
```
1. 锚点导航条（申万行业/概念板块切换）
2. 板块轮动速度卡片
3. 申万行业区域：
   - 行业涨跌幅热力图
   - 行业搜索条
   - 行业指数网格
4. 概念板块区域
```

### 建议顺序
```
1. 板块轮动速度卡片          ← 先判断行情性质：趋势还是电风扇
2. 行业涨跌幅热力图          ← 再看哪些行业在涨/跌
3. 锚点导航条（申万行业/概念板块切换）
4. 行业搜索条
5. 行业指数网格              ← 具体行业详情
6. 概念板块区域
```

### 涉及文件
- `web/app.js` — `renderIndustry()` 函数
- `static-site/app.js` — 同上

## 四、不调整的部分

以下页面/模块保持现有顺序不变：
- **概览页 KPI 卡片**：用户已优化过，不调整
- **概览页两列卡片**：已按用户要求排列（冰点日+买卖点，均线排列+位置感）
- **港股Tab**：内容少（港股通净买入 + 指数K线），不需要重排
- **全球Tab**：商品/债券/汇率按品类分组，已合理
- **市场Tab 更多折叠区**：龙虎榜/解禁/IPO/可转债，低频参考，折叠区无需排序
